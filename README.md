A Thousand Counts
=================
_www.athousandcounts.com_

A Thousand Counts is creating a video of 1000 different people counting to 1000.

The site is made with the [_ASP.NET-MVC 5_](http://www.asp.net/mvc) framework and makes use of the [_ScriptCam plugin_](https://www.scriptcam.com/) and [_Metro-UI CSS_](http://metroui.org.ua/). A Thousand Counts is hosted on [_Microsoft Azure_](https://azure.microsoft.com/en-us/),
